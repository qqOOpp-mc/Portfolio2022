꾸쁘의 시스템 개발 작업물

최적화된 알고리즘을 기반으로
군더더기 없이 빠른 시스템을 개발합니다.

사용 가능 툴

  • IntelliJ
  • Eclipse
  • Visual Studio Code





<GUI 및 오버레이>

마인크래프트 내 GUI와 오버레이를 자유자재로 다룰 수 있습니다.






          

<휴대폰 GUI>

특정 아이템 우클릭시 왼쪽의 연락처 목록이 나오고,
연락처 목록에서 인물 선택 시
선택한 인물과 음성통화를 진행할 수 있습니다.

(Linmalu VoiceChat 플러그인과 연동)






<엘레베이터 GUI>

특정 블록 우클릭 시 위와 같은 GUI가 출력되고
몇층으로 이동할지 선택 하면 효과음과 함께
해당 층으로 이동합니다.






<로빈 유튜브 - 블럭변신>

모든 베드락 몹, 조합법, 베드락 건틀렛,
베드락 바주카포 등을 작업했습니다.






<로빈 유튜브 - 집으로 가게>

오징어 굽기, 국밥 그릇에 담기, 소스 바르기 등
조리 시스템을 개발하였습니다.






<로빈 유튜브 - 후즈유어대디>

레고 어지르기, 장난감 총, 달걀 던지기,
젓가락으로 콘센트 찌르기 등의 기능을 개발하였습니다.
